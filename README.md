# donkey-vis
Donkeycar neural network model visualization built on top of https://github.com/raghakot/keras-vis

![example](https://github.com/markku-ai/donkey-vis/blob/master/example.gif)


## Requirements
1. OpenCV
1. Conda
1. Trained neural network
1. Donkeycar records tub

## Usage
1. Install opencv (OS Specific)
1. Install conda environment `conda env create -f environment.yml`
1. Activate conda env `conda activate donkey-vis`
1. For now: `pip install git+https://github.com/raghakot/keras-vis.git --upgrade` (as long as some bug has been fixed)
1. Check that the installed tensorflow version matches with the version model was trained on `conda list | grep tensorflow` (Defaults to 1.11.0)
1. If it doesn't match, update tensorflow version `conda install tensorflow-gpu==1.11.0`
1. Run visualization with `python visualize --model ./model --tub ./records` (`python visualize -h for more parameters`)
1. Kill with ctrl-c when enough frames exported
